Использованные инструменты:
Python 3.10
Ubuntu 20.04.4
PyCharm 2021.3

Использованные библиотеки:
socket - используется для создания сокета для связи между сервером и клиентами 
threading - используется для одновременной поддержки нескольких клиентов в разных потоках
syslog - используется для логгирования сообщений в Unix-системах
sys - используется для выхода из системы с помощью sys.exit()

Для создания дистрибутивов была использована библиотека pyinstaller
$ pyinstaller --onefile <filename>

Распаковка
Для распаковки загрузите дистрибутив с https://github.com/rogaliiik/console_chat.git либо загрузите архив
Разархивируйте пакет $ tar -xvf console_chat.tar -C /path/to/dir

Перейдя в папку /dist, в консоли запустите сервер с помощью команды $ ./server 
Далее укажите необходимые хост и порт, в случае использования настроек по умолчанию пропустите ввод.
После для соединения клиента с сервером из директории /dist запустите $ ./client
Укажите необходимые  хост и порт (для соединения с сервером необходимо ввести хост и порт сервера)
После подключения чат полностью доступен для использования.
Для выхода введите /exit.
